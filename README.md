# motion-flick
Motion Flick is an app which get data from The Movie DataBase (TMDB) to fetch Trending and Top Rated movies list.

# Applied technologies and libraries:
1) Android X
2) MVVM Design Pattern
3) Hilt (Dagger)
4) Retrofit 
5) Moshi
6) Okhttp
7) Glide
8) Flow
9) Kotlin Coroutines
10) Room Database
11) UI in Jetpack Compose
12) Material Design
